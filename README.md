## In case of Mac

## Installation

```bash
$ git clone https://github.com/hinatha/lab-efficient-system.git
$ cd lab-efficient-system/api
$ . .venv/bin/activate
(.venv) $ python3 -m pip install -r requirements.txt
```

## Usage

```bash
(.venv) $ python app.py
```
